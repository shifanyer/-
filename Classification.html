Классификация компьютерных систем

Типы компьютеров:

  • Настольные компьютеры (desktop)
  • Многоцелевые компьютеры (mainframe)
  • Суперкомпьютеры
  • Кластеры компьютеров (Computer clusters)
  • Портативные компьютеры (Laptop)
  • КПК (PDA)
  • Мобильные устройства (mobile intelligent devices)
  • Носимые компьютеры (Wearable computers)
  • Распределённые системы (distributed systems)

FLOPS - внесистемная единица, используемая для измерения производительности компьютеров, показывающая, сколько операций с плавающей запятой в секунду выполняет данная вычислительная система.

Первые суперкомпьютеры:

  1. ЭНИАК (США) 500 флоп
  2. БЭМС-2 (СССР) 10 кфлоп
  3. IBM (США) 5 кфлоп

Для сравнения в 2014 году Intel Core i7 (Extreme Edition) - 350 гигафлопсов
