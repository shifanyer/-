Современные операционные системы

Операционные системы используются везде и всюду.

Классификация операционных систем происходит согласно функциям, на которые они ориентированы.

Классы ОС:

  1. ОС для персональных компьютеров
  2. Встраиваемые системы
  3. UNIX (предназначены для работы на мощных компьютерах предприятий)
  4. UNIX-подобные (предназначены для работы на серверах)
  5. ОС для мейнфреймов

Что отличает современную ОС от старой:

  • Ориентация на сетевой ввод-вывод
  • Конвергенция решаемых задач
  • Уклон в сторону контейнерной виртуализации

Системы по типу ядер:

  • С монолитным ядром
  • С гибридным ядром
  • С модульным ядром
  • С микроядром
  • С наноядром
  • С ещё меньшими размерами ядер

В монолитном ядре:

Все программы - компоненты тесно связаны друг с другом. Из системы нельзя убрать какой-либок омпонент, т.к. ядро перестанет функционировать

В гибридном ядре:

Часть компонентов реализовано при помощи подключаемых модулей, а часть - всё ещё являются частью ядра.

В модульном ядре:

Основная функциональность вынесена в подключаемые модули. В центральной части оставлены только самые важные компоненты.

В микроядрах (и меньше):

Всё стараются перенести на подключаемые модули.
